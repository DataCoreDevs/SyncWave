# SyncWave

**SyncWave** is a mood-based music recommendation web app that personalizes song suggestions based on user mood and listening behavior.  
The project focuses on building a dynamic user experience with real-time interaction, mood analysis, and adaptive recommendations.
It simulates an intelligent music system by combining user interaction tracking with ML based predictions.  
It adapts recommendations by daily based user behavior and mood patterns.

##  Features  

-  Mood-based song recommendations  
-  Mood analysis using recent listening history (≥5 songs)  
-  Adaptive recommendations with weighted mood bias  
-  Recent plays tracking (last 25 songs)  
-  14-day mood history tracking  
-  Manual mood override option  
-  Dynamic UI with real-time updates
