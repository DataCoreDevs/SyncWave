# SyncWave

**SyncWave** is AI powered mood-based music recommendation web app that personalizes song suggestions based on user mood and listening behavior. 
It recommends music based on current mood and predicts future mood using deep learning — trained on **89,741 real Spotify tracks.
The project focuses on building a dynamic user experience with real-time interaction, mood analysis and adaptive recommendations.
It simulates an intelligent music system by combining user interaction tracking with ML based predictions.  
It adapts recommendations by daily based user behavior and mood patterns.

##  Features  

-  Mood-based song recommendations | Songs served from K-Means mood clusters (89,741 tracks)
-  Mood analysis using recent listening history (≥5 songs) |  LSTM predicts next mood from last 5 sessions 
-  Adaptive recommendations with weighted mood bias  
-  Recent plays tracking (last 25 songs)  
-  14-day mood history tracking  
-  Manual mood override option | User can switch mood anytime 
-  Zero-friction storage | Fully LocalStorage — no account needed 
-  Spotify player | Embedded Spotify iframe for instant playback 
-  Dynamic UI with real-time updates
